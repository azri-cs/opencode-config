# DevOps and Deployment

Apply this skill when infrastructure and delivery workflows are central to the task.

## Focus areas

- Build and deployment pipelines
- Container and runtime configuration
- Environment promotion and rollback strategy
- Observability, alerts, and recovery
- Secrets, permissions, and operational safety

## Workflow

1. Prefer reproducible automation over manual steps.
2. Make environment differences explicit.
3. Add health checks, rollback paths, and observability.
4. Treat secrets and credentials as first-class security concerns.
5. Call out deployment blast radius and recovery steps.

## Deliverables

- Pipeline or deployment guidance
- Infrastructure safety recommendations
- Monitoring and rollback notes
- Suggested validation before and after release
