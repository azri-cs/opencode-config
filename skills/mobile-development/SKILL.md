# Mobile Development

Apply this skill when the task targets mobile app behavior or architecture.

## Focus areas

- App structure and navigation
- Native integrations and permissions
- Offline and synchronization behavior
- Performance on constrained devices
- Mobile-specific UX and accessibility

## Workflow

1. Design around real device constraints first.
2. Keep navigation, state, and data sync predictable.
3. Plan for intermittent connectivity and retries.
4. Minimize battery, memory, and startup cost.
5. Verify platform-specific edge cases explicitly.

## Deliverables

- Mobile architecture guidance
- Integration and offline behavior notes
- Performance and UX considerations
- Suggested device and scenario testing
