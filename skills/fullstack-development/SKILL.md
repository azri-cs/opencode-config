# Fullstack Development

Apply this skill when work crosses multiple layers of the application.

## Focus areas

- End-to-end feature slicing
- Contract alignment between UI and backend
- Data flow from storage to presentation
- Error handling across boundaries
- Delivery sequencing and verification

## Workflow

1. Define the user-visible behavior first.
2. Identify backend, data, and UI changes needed to support it.
3. Keep contracts explicit between layers.
4. Prefer incremental delivery with clear checkpoints.
5. Verify the happy path and critical failure paths.

## Deliverables

- End-to-end implementation plan
- Layer-by-layer change breakdown
- Contract and data-flow notes
- Suggested integration and regression tests
